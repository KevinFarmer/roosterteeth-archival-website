# Archive of Pimps

## Add links

Edit `src/data/archives.json`

Format is:

```json
{
  "category": "Name of section",
  "links": [
    {
      "name": "Name of archive",
      "url": "LINK_TO_ARCHIVE"
    }
  ]
}
```

## Run dev website

`npm install`

`npm run dev`

This website is using [Astro](https://astro.build) for the framework and [Svelte](https://svelte.dev) for any Javascript.
